# FrontWars Clone

A high-fidelity multiplayer real-time strategy game built with React, Vite, and Socket.io.

## Features
- **Real-time Multiplayer:** Fast-paced territory conquest with Socket.io.
- **Tactical Gameplay:** Capture bases, manage troops, and dominate the map.
- **Modern UI:** Sleek, military-themed interface with Rajdhani typography.

## Run Locally

**Prerequisites:** Node.js

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the server:**
   ```bash
   cd server
   npm install
   npm start
   ```
3. **Run the client:**
   In a new terminal:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to play.
